## 🎯 User Flows

### Patient Journey:
```
1. Land on home page
2. Click "Register" → Sign up as patient
3. Redirected to dashboard
4. Click avatar → "View Profile"
5. Fill out profile information
6. Save changes → Success toast
7. Navigate to "Find a Specialist"
8. Use filters to find doctor
9. Click "View Profile" on doctor card
10. Book appointment (future)
```

### Doctor Journey:
```
1. Land on home page
2. Click "Register" → Sign up as doctor
3. Provide medical license
4. Redirected to dashboard
5. Click avatar → "View Profile"
6. Complete professional profile
7. Set consultation fees
8. Configure availability
9. Save changes → Success toast
10. Wait for verification
11. Start receiving appointments (future)
```
